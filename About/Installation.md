# Installation

## Pip

You may install MCGpy directly with pip. We recommend this way.

```sh
$ pip install mcgpy
```

## Requirements

MCGpy requires the following Python packages:

| Name     | Version     | 
|----------|-------------|
| numpy    |  `>=1.2.0`  |
| astropy  |  `>=3.0.0`  | 
| scipy    |  `>=1.2.0`  | 
| h5py     |  `>=2.7.0`  |

They will be automatically installed when you install MCGpy.
